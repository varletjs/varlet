# varlet eslint config
varlet eslint 配置文件

## 快速上手
```shell
# npm安装
npm i eslint @varlet/eslint-config -D

# yarn安装
yarn add eslint @varlet/eslint-config -D

# pnpm安装
pnpm add eslint @varlet/eslint-config -D
```

添加如下代码到package.json
```
"eslintConfig": {
  "root": true,
  "extends": [
    "@varlet"
  ]
}
```
