# varlet stylelint config
varlet stylelint 配置文件

## 快速上手

```shell
# npm安装
npm i stylelint @varlet/stylelint-config -D

# yarn安装
yarn add stylelint @varlet/stylelint-config -D

# pnpm安装
pnpm add stylelint @varlet/stylelint-config -D
```

添加如下代码到package.json
```
"stylelint": {
  "extends": [
    "@varlet/stylelint-config"
  ]
}
```
