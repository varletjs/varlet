# varlet-cli-app

### Quickstart

```shell
yarn
yarn dev
```

### MenuTypes

```ts
enum MenuTypes {
  TITLE = 1,
  COMPONENT = 2,
  DOCUMENTATION = 3
}
```

### Commands

#### Start the development server

```shell
yarn dev
```

#### Build documentation site

```shell
yarn build
```

#### Building component libraries

```shell
yarn compile
```

#### Run unit test

```shell
yarn test
```

#### Run unit test in watch mode

```shell
yarn test:watch
```


#### Quickly create a component folder

```shell
yarn run create <componentName>
```

#### Mount git hook

```shell
yarn husky
```

### Publish

```shell
yarn publish
```
