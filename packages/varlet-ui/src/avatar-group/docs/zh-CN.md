## API

### 属性

| 参数                  | 说明                                        | 类型   | 默认值          |
|---------------------|-------------------------------------------|------|--------------|
| `offset`          | 头像组内的头像偏移量	| _number \| string_ | `-` |
| `vertical`        | 组内头像是否垂直排列 | _boolean_ | `false` |

### 插槽

| 名称 | 说明 | 参数 |
| ----- | -------------- | -------- |
| `default` | 头像组内容 | `-` |
