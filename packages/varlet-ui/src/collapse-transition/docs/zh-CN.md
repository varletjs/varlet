## API

### 属性

| 参数       | 说明                     | 类型        | 默认值    |
| ---------- | ------------------------ | ----------- | --------- |
| `expand` | 是否展示内容             | *boolean* | `false` |
| `appear` | 是否在首次出现时播放动画 | *boolean* | `false` |

### 插槽

| 名称        | 说明       | 参数 |
| ----------- | ---------- | ---- |
| `default` | 折叠的内容 | -    |
