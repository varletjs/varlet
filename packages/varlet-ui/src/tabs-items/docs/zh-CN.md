## API

### 属性

| 参数 | 说明 | 类型 | 默认值 | 
| --- | --- | --- | --- | 
| `v-model:active` | 激活的选项卡的名字 | _string \| number_ | `-` |
| `can-swipe` | 是否允许滑动切换 | _boolean_ | `true` |
| `loop` | 是否允许循环切换 | _boolean_ | `false` |

### 插槽

| 插槽名 | 说明 | 参数 |
| --- | --- | --- |
| `default` | 视图组的内容 | `-` |

### 方法

| 方法名     | 说明              | 参数 | 返回值 |
| ---------- |-----------------| ---- | ------ |
| `getSwipe` | 获取 `swipe` 组件实例 | `-`  | `-`    |
