# 设计资源

### 介绍

我们为设计师提供了设计资源，您可以在本页面选择所需的资源进行下载。

### 组件资源

组件资源包含了`色彩`、`文字`、`组件资源`。目前只提供 Material Design 2 亮色模式的设计资源，其他资源暂未提供，欢迎社区贡献。
